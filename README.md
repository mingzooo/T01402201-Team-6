# DONT SET HIM OFF (HUFS 게임프로그래밍 Team6)

## 👩‍💻 About Project  

2023-1 한국외대 게임프로그래밍 6팀 기말고사 과제입니다.  
존윅 1 스토리를 모티브로 자신이 아끼는 개를 죽인 악당들에게 복수하며 펼쳐지는 2D 횡스크롤 슈팅 액션 게임입니다.

## 👀 Character
